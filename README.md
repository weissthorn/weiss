- [NodeJS 18 LTS upward](https://github.com/nvm-sh/nvm/blob/master/README.md)
- [RethinkDB 2.4 upward](https://rethinkdb.com/docs/install/)


```
yarn dev
```

### Production

```
yarn production
```
