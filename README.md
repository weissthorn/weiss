### Prerequisite

- [NodeJS 12 LTS upward](https://github.com/nvm-sh/nvm/blob/master/README.md)
- [RethinkDB 2.4 upward](https://rethinkdb.com/docs/install/)
- Nginx
- Apache server 2.4

### Installation

```
npm i
```

### Goal

- User system (member & moderator)
- Social login (Facebook, Google, Github, Twitter)
- Communities or categories
- Post and comment
- Social share
- Email notification
- Web notification
- Translation (English, Arabic, Spanish etc)
- Coin and reward
